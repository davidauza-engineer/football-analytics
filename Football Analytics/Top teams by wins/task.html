Description

In the first stage of our football analytics project, you will focus on setting up the environment by downloading and extracting the necessary data files. You will work with a SQLite database containing extensive football match data.

You might have noticed that there is no accompanying ERD provided with the data. This adds an extra layer of excitement to working with the database. You can use tools like DbVisualizer to generate the ERD and query the database. You can generate the ERD using the following steps.

By the end of this stage, you will have successfully retrieved the database and performed a simple analysis to identify the top 10 teams with the most wins using SQL and Python.

Objectives

To complete this stage:

  1. Download the database.zip file and save it in the data directory.
  2. Extract the content of the zip file to the data/database directory.
  3. Connect to the SQLite database file named database.sqlite with sqlite3.
  4. Perform the analysis to count the number of wins for each team and list the top 10 teams by wins using SQL query and the Pandas library. Using Pandas' read_sql_query function may be important for running your SQL query.
  5. Print the results as a list of lists.
