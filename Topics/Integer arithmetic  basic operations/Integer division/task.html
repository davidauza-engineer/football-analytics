Let's tackle this question now: how can we obtain an integer by dividing two numbers?

The answer is the integer division (also known as floor division and denoted by \(//\)) in Python. While regular division (\(/\)) yields a floating-point number, integer division produces an integer value by discarding the decimal part. Here's a quick example to illustrate the contrast:

print(7 / 2)
print(7 // 2)

The first line will print \(3.5\) but the second line will print only the integer part, which is \(3\).
