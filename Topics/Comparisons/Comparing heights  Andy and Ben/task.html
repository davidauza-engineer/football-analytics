Write a Python program that reads two integer numbers from the input. The first integer is the height of Andy, and the second one is the height of his best friend Ben. Your program should print 'True' if Andy is taller than Ben or 'False' otherwise.


Sample Input:
150
160

Sample Output:
False


Sample Input:
170
165

Sample Output:
True



Memory limit: 256 MB
Time limit: 15 seconds
