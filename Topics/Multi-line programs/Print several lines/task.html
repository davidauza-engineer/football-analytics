You have learned how to write simple, single-line Python programs that print text. However, real programs contain many lines. You can write such programs using the print() function that you are already familiar with.

Let's dive into an example of using the print() function to print multiple lines. Check this out:

print("I")
print("know")
print("Python")

This code prints three lines:

I
know
Python

Pretty cool, right? Now it's your turn! Try printing the following text using Python:

There is a cat on the keyboard, it is true,
Typing messages out of the blue.

Please pay attention to the cases and punctuation marks. You must print the two lines exactly as they are.
