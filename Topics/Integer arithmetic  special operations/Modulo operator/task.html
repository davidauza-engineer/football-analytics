In the last step, you tackled the remainder calculation with three operators. But Python is all about simplicity. The modulo operator (\(\%\)) â€”it's here to streamline your life. With Python's modulo operator (\(\%\)), you can easily find the remainder of any division.

One of its handy uses is checking whether a number is even or odd. For instance:

print(25 % 2)

The output is \(1\), which is the remainder when we divide \(25\) by \(2\), indicating that \(25\) is an odd number.

print(20 % 2)

This time, the output is \(0\) because \(20\) is an even number. So there is no remainder.
