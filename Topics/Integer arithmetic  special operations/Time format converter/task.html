We've stored a time value in the 24-hour format in a variable named hour. Your task is to convert this value to a 12-hour format and print the result.

Note that you should only print the hour as an integer. Do not include AM or PM in your output.

Example:

  • If hour = 14, the output should be 2.

  • If hour = 0, the output should be 12.

You can solve this using the modulo operator % or basic arithmetic. The goal is to help you understand how to convert time formats using simple math.
