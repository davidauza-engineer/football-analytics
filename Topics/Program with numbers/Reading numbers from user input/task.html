Since you have become familiar with the input() function in Python, it's hardly new to you that any data passed to this function is treated as a string.

But how should we deal with numerical values? As a general rule, they are explicitly converted to corresponding numerical types. To do it, we add data type before input:

integer = int(input())
floating_point = float(input())
