Python won't let you confuse it by using variable names that match its built-in keywords. So, you can't name your variable with any of these reserved words.

You can check out the full list of reserved names here: Python Reserved Keywords. We are also providing the screenshot of the list for your convenience:

Hence, you must avoid naming your variable using any words from the list above. For example:

# These are wrong variable names
False = 0
try = 5
print(global)

Each line above will throw an error.

And, here are some good examples:

# These are good variable names
is_true = 0
max_items = 5
international = "United Nations"
print(international)
