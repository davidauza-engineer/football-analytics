It might be nighttime where you are, so "Good Morning!" might not be quite right. No worries, you can easily change the message to "Good Evening!" by assigning a new value to the variable.

For example:

message = "Good Evening!"

Now, the message variable holds "Good Evening!" instead. Hope that brings a smile to your face!
