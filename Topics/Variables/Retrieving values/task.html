Your friend might take forever to pay you back, but Python is way more reliable. You can grab the data from a variable whenever you need it. How? Just call the variable by its name!

For example, if you write:

print(message)

Python will gladly print out the value stored in message, which is "Good Morning!". Easy peasy!
