Heads up for an important rule!

Imagine you have two kitties named Tic and Tac, and you want to store their names in a variable. You can do it like this:

two_kitties = "Tic, Tac"

That's perfectly fine. But what if you're feeling a bit lazy and decide to name the variable like this:

2_kitties = "Tic, Tac"

Nope! Python doesn't allow variable names to start with a number.

Remember, no starting a variable name with a number. Let's keep that rule in mind always!
