Introducing the first data type in Python: the string. Strings are the superheroes of text handling in programming. In Python, we shorten it to "str." These data dynamos have some pretty cool tricks up their sleeves.

One of the coolest? You can wrap strings in either single quotes (' ') or double quotes (" "). Python's doesn't care about whichever one you choose! (Well, brace yourself for a twist in the next step.)

For instance:

print("Hello world!")
print('Hello world!')

Both of these will print Hello world!.
