You've already seen that the type() function returns <class 'int'> for integer numbers. Now, we just have the float type of numbers left to explore. Let's dive in:

print(type(3.14))

And voila! It spits out:

<class 'float'>

Now, here's a brain teaser for you: What do you think will be the output of the following line of code?

print(type(-30))
